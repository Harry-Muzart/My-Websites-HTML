    Based on the topics in the menu above, I aim to construct & build data-driven Computational Models (3D Graphical Representations, Functional Computational Models, Web-based Scripts, Apps, Programs for Analytics/Informatics, Relational Databases & Big-Data Networks, etc) either from scratch using base hard code or using existing platforms.

     

    I deal more with prototyping in software engineering, computing and virtual simulation models; but also some basic hardware implementation and physical experimentation.

     

    I focus on biosciences/neurosciences at all levels of organisation (esp. at low-level and high-level cellular network systems), and I look at how this can be merged with technology, especially based on compatible computational architectures.

    My key aim is to understand complex systems.


    There are a few focus points from which this site originally started:

    • Exploring the interdisciplinary link between biology and technology. 

    • Using a computational and neuroscientific mindset to modelling & analysing complex systems.

    • Solve computational neuroscience and its medical and enhancement applications.

    • Solving humankinds other problems.

     

    I have deviated from this a bit and have extended my scope to include other fields.


    As said, my core work is science & tech research in each of these topics, and projects focus on not only coding visual representations but also working on hard-coding code for functional computational interactomics models that effectively use and process data. My aim is broadly applying STEMM principles to work out mechanisms, and innovating fundamental core code models. There is also the producing of other products in various formats, such as big-data files, executable scripts, or packaged apps. There are also other further functional applications.

    The web platform aims to develop useful tools to help us solve problems. It is based on using practical applications of 4 main fields:

    --1-- Neurobiology.

    --2-- Cognitive Sciences. 

    --3-- Computer Engineering.
    --4-- Computational Neuroscience, Neural Engineering, Biological Informatics,

    Deep Machine Learning, and Cognitive Analytics. 

    Various projects are currently being worked on, centered around the idea that we need to be using computational neuroscience to further our understanding of (and progress in) computational neuroscience itself. These are inspired by current advances being developed by various STEMM-based private firms.
     

    My long-term aim is of: capitalising from the research & development, monetising fully this site's products, making collaborations, attracting investors, having a foundation for post-doctoral work, etc.

     

    There will be various vision milestones that will be aimed for: eg.

    --- Help solve intelligent systems

    --- Applying this to pattern recognition of fMRI/dtMRI tractography scans and connectome data

    --- Applying this to models of regeneration of CNS neurons
